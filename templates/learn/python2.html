โครงสร้างควบคุม

เงื่อนไข if-else

โครงสร้างพื้นฐาน if condition: # ทำอะไรบางอย่าง elif another_condition: # ทำอย่างอื่น else: # ทำเมื่อไม่ตรงเงื่อนไขใดๆ

ตัวอย่าง: ```python age = 18 if age < 13: print("เด็ก") elif age < 18: print("วัยรุ่น") else: print("ผู้ใหญ่") ```

การเปรียบเทียบ

- `==`: เท่ากับ - `!=`: ไม่เท่ากับ - `>`: มากกว่า - `<`: น้อยกว่า - `>=`: มากกว่าหรือเท่ากับ - `<=`: น้อยกว่าหรือเท่ากับ

ตัวดำเนินการทางตรรกะ

- `and`: และ - `or`: หรือ - `not`: ไม่

```python x = 5 y = 10 if x > 0 and y > 0: print("ทั้ง x และ y เป็นบวก") if x > 0 or y > 0: print("x หรือ y อย่างน้อยหนึ่งค่าเป็นบวก") if not x < 0: print("x ไม่น้อยกว่า 0")

ลูป (Loops)

for loop

```python # วนลูปจาก 0-4 for i in range(5): print(i)

# วนลูปผ่านสมาชิกของลิสต์ fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit)

loop ผ่าน dictionary person = {"name": "John", "age": 25, "job": "Developer"} for key, value in person.items(): print(f"{key}: {value}")

while loop

```python # วนลูปจนกว่าเงื่อนไขจะเป็นเท็จ count = 0 while count < 5: print(count) count += 1 ```

การควบคุมลูป

- `break`: ออกจากลูปทันที - `continue`: ข้ามการทำงานที่เหลือในรอบนี้และเริ่มรอบถัดไป - `pass`: ไม่ทำอะไร (placeholder)

# ตัวอย่างการใช้ break for i in range(10): if i == 5: break print(i) # จะพิมพ์ 0, 1, 2, 3, 4 # ตัวอย่างการใช้ continue for i in range(10): if i % 2 == 0: continue print(i) # จะพิมพ์ 1, 3, 5, 7, 9 # ตัวอย่างการใช้ pass for i in range(5): if i == 2: pass # ไม่ทำอะไรเป็นพิเศษ print(i) # จะพิมพ์ 0, 1, 2, 3, 4

List Comprehension

เป็นวิธีการสร้างลิสต์ใหม่จากลิสต์ที่มีอยู่อย่างกระชับ: ```python # สร้างลิสต์จากเลข 0-9 numbers = [x for x in range(10)] # สร้างลิสต์เฉพาะเลขคู่ even_numbers = [x for x in range(10) if x % 2 == 0] # สร้างลิสต์ของกำลังสอง squares = [x**2 for x in range(10)]
