ตัวแปรใน Python

ตัวแปรใน Python สามารถประกาศได้โดยไม่ต้องระบุชนิดข้อมูล:


# การประกาศตัวแปร
name = "John"
age = 25
is_student = True

ประเภทข้อมูลพื้นฐาน

1. ตัวเลข (Numbers)

  • int: จำนวนเต็ม เช่น 1, 100, -10
  • float: จำนวนทศนิยม เช่น 3.14, -0.5
  • complex: จำนวนเชิงซ้อน เช่น 1+2j

x = 5          # int
y = 3.14       # float
z = 2 + 3j     # complex

2. สตริง (String)


name = "Python Tutorial"
char = 'A'
multiline = """This is a
multiline string"""

การเชื่อมสตริง:


first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name  # "John Doe"

การทำ String Formatting:


name = "John"
age = 25
message = f"{name} is {age} years old"

3. บูลีน (Boolean)


is_true = True
is_false = False

4. ลิสต์ (List)


fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", True, 3.14]

การเข้าถึงสมาชิกในลิสต์:


fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # "apple"
print(fruits[1])  # "banana"

5. ทูเพิล (Tuple)


coordinates = (10, 20)
colors = ("red", "green", "blue")

6. ดิกชันนารี (Dictionary)


person = {
    "name": "John",
    "age": 25,
    "is_student": True
}

การเข้าถึงข้อมูลใน dictionary:


person = {"name": "John", "age": 25}
print(person["name"])  # "John"
print(person["age"])   # 25

7. เซต (Set)


fruits = {"apple", "banana", "cherry"}

การตรวจสอบประเภทข้อมูล


x = 5
print(type(x))  # 

name = "John"
print(type(name))  #

การแปลงประเภทข้อมูล


# แปลงเป็น int
x = int(3.14)    # x = 3
y = int("10")    # y = 10

# แปลงเป็น float
a = float(5)     # a = 5.0
b = float("3.14")  # b = 3.14

# แปลงเป็น string
s = str(10)      # s = "10"
t = str(3.14)    # t = "3.14"
