บทที่ 1: ประเภทข้อมูล (Data Types) ใน Python

Python คืออะไร

Python เป็นภาษาการเขียนโปรแกรมที่ได้รับความนิยมสูง เนื่องจากมีโครงสร้างที่อ่านง่าย ใช้งานง่าย และมีชุมชนที่ใหญ่ Python สามารถใช้สำหรับการพัฒนาเว็บ, วิเคราะห์ข้อมูล, ปัญญาประดิษฐ์ (AI), และการพัฒนาเกม เป็นต้น

ประเภทข้อมูล (Data Types) ใน Python

Python มีประเภทข้อมูลหลักๆ ดังนี้

  • int: จำนวนเต็ม เช่น 1, 2, 3
  • float: จำนวนทศนิยม เช่น 1.0, 2.5, 3.14
  • str: ข้อความ เช่น "สวัสดี ชาวโลก!"
  • bool: ค่าความจริง เช่น True, False
