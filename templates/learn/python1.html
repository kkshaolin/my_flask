ตัวแปรและชนิดของข้อมูล

ก่อนเริ่มเรียนเรามารู้จักฟังก์ชัน print() กันก่อน ฟังก์ชันที่ใช้ในการแสดงค่าตัวอย่างเช่น เมื่อใช้ print("Hello World") จะแสดงผลลัพธ์ เป็น Hello World ออกมา

ทดลองเขียนดูแล้วกด run ผลลัพท์ที่ต้องการคือ Hello World!

ตัวแปร

"ตัวแปร" คือสิ่งที่ใช้เก็บค่าของข้อมูลชนิดต่างๆ นั่นเพราะเวลาที่เรามีข้อมูลก็จำเป็นต้องมีที่จัดเก็บข้อมูล ซึ่งที่จัดเก็บข้อมูลนั้นก็คือตัวแปรนั่นเอง ถ้าให้เปรียบเทียบก็อาจเทียบว่าตัวแปรหนึ่งตัวคือกล่องที่เก็บข้อมูลไว้

ชนิดของข้อมูล

ในตารางเป็นแค่ส่วนหนึ่งไม่ใช้ทั้งหมด

ชนิด ตัวย่อใน python ตัวอย่าง
จำนวนเต็ม int 123
จำนวนทศนิยม float 12.3
ค่าความจริง bool False
ข้อความ str "Hello World"
ลิสต์ list [1, 2, 3]
ทูเพิล tuple (1, 2, 3)
เซต set {1, 2, 3}
พจนานุกรม dict {"name": "A", "age": 25}

โดยในการเปลี่ยนชนิดข้อมูลใน python คือการใส่ตัวย่อหน้าข้อมูลแล้วใช้วงเล็บล้อมรอบข้อมูล เช่น int(1.23) = 1

ทดลอง เปลี่ยน 123 เป็น list โดยเก็บในตัวแปล แล้วใช้ print() ดู ผลลัพท์ที่ต้องการคือ [123]
